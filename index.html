Welcome to ToDo:Read home page.

What is ToDo:Read?

ToDo:Read is application for reading articles you have postponed.

Support for Read It Later service enables you to read articles anytime from your mobile device.

This application also allows you to download the articles for reading offline.

Platforms

ToDo:Read currently runs only on Meego/Harmattan (Nokia N9 and Nokia N950).

How to use it

For first steps visit Getting started page on wiki.

Where can I get it?

You can grab .deb package and install it to your smartphone. This application is now also on the way to Ovi store.
